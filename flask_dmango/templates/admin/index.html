1. 소개

MongoDB를 이용해 자유롭게 RESTful 방식으로 데이터를 저장/조회 해보세요.

no code, no script !

dmango는 복잡한 설정이 필요하지 않습니다. mongodb 접속정보만 넣어줘도 OK.

예약어 안내 : $sort , $pagenum, $pagesize

2. 사용법

필요한 기능을 다음과 같이 호출해서 사용하세요.

# 기능 URI
1 안내페이지(현재페이지) {{prefix}}
2 파라미터테스트 {{prefix}}/_test?...
3 Insert {{prefix}}/{collection}/insert?...
4 Remove {{prefix}}/{collection}/remove?_id={_id}
{{prefix}}/{collection}/remove?_id={_id1}&_id={_id2}
5 Update {{prefix}}/{collection}/update?_id={_id}&...
6 Push {{prefix}}/{collection}/push?_id={_id}&...
7 Find {{prefix}}/{collection}/find?...
8 List(html) {{prefix}}/{collection}/list?...
{{prefix}}/{collection}?...
